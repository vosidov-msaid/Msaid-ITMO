1. mkdir - создание каталога
- mkdir <name_folder>
* `-p` - создать родительские каталоги

2. echo - вывод текста
- echo <строка>
* -e - интерпретировать спецсимворы (\n, \t)
* -E - отключить интерпретацию спецсимволов

3. cat - вывод содержимого файла
- cat <filename>
* -n - номеровать все строки
* -b - номеровать непустые строки
* -s - удалить повторяющиеся пустые строки

4. touch - создать новый пустой файл или обновить время доступа
- touch <file>

5. ls - вывод списка файлов и папок
* -l - детальная информация
* -a - показать скрытые файлы
*** фильтрация по части имени
* ls *.txt - только txt файлы
* ls | grep 'test' - только те, что содержат "test" в имени

6. pwd - показать текущий рабочий каталога
- pwd

7. cd - сменить каталог
- cd <path_to>

8. more - постраничный просмотр файла
- more <file>

9. cp - копировать файлы и каталоги
- cp [options] <from> <to>
* -r - рекурсивно (для директории)
* -i - запрос подтверждения при замене

10. rm - удалить файлы и каталоги
- rm [options] <file>
* -r - рекурсивно для каталога
* -f - принудительно
* -i - запрос на удаление

11. rmdir - удалить пустой каталог
- rmdir <folder>

12. mv - переместить или переменовать файл/каталог
- mv [options] <from> <to>

13. ln - создание ссылок внутри дерева
- ln original_file link_file
* -s - указывает, что это символьная ссылка

14. cat - скопировать содержимое файла в другой
- cat file1 > file2
* весь текст file1 перезаписывает в file2

- cat file1 >> file2
* добавить весь текст file1 к концу file2

- cat file1 file2 file3 > summary.txt
* В summary.txt объединены все указанные файлы

15. wc - подсчитать количество строк, слов и символов
- wc file.txt - количество строк и слов
- wc -l file.txt - количество строк

16. head и tail - вывод первых/последних строк файла
- head file.txt - первые 10 строк
- head -5 file.txt - первые 5 строк
- tail file.txt - последние 10 строк
- tail -3 file.txt - последние 3 строк

17. sort - сортировка данных из файла/каталога
- sort file.txt - отсортировать строки файла
- ls | sort - отсортировать имена файлов

18. grep - поиск по содержимому файлов
- grep 'фраза' file.txt - только строки с "фраза"
- grep -i 'фраза' file.txt - без учёта регистра
- grep -r 'TODO' . - рекурсивно во всех файлах текущего дерева

19. Посчитать количество найденных строк
- grep 'word' file.txt | wc -l

20. 2>&1 - перенаправляет поток ошибок в поток вывода
